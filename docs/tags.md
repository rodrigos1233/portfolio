# Controlled Tag Vocabulary

Tags are validated at build time. Use lowercase, hyphenated names.

## Categories

### Domain
- `web` — Web applications
- `api` — API services / backends
- `cli` — Command-line tools
- `infrastructure` — Infrastructure, DevOps, platform
- `developer-tools` — Tools for developers
- `audio` — Audio processing / music
- `data` — Data processing / pipelines

### Pattern
- `real-time` — Real-time / WebSocket features
- `visualization` — Data visualization
- `graph` — Graph structures / network analysis
- `sdk` — Library / SDK

### Scope
- `open-source` — Open source project
- `production` — Running in production
- `prototype` — Proof of concept / prototype

### Technology
- `web-api` — Uses browser Web APIs (Web Audio, Canvas, etc.)
